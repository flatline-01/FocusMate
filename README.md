# FocusMate

A task manager programme written in C# with a Pomodoro timer.

## Using Guide

1. Download [the last release](https://github.com/flatline-01/FocusMate/releases/tag/1.5.0)

2. Install PostgreSQL and create a database.

3. Download [focus_mate.sql](https://github.com/flatline-01/FocusMate/blob/master/focus_mate.sql)

4. Import focus_mate.sql to your database 

> psql -h hostname -d databasename -U username -f file.sql

5. Launch the programme. Enter your PostgreSQL username and password.
